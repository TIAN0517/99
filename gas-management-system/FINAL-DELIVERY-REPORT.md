## 🚀 部署说明

### VPS 部署步骤
1. 上传 `gas-management-system-vps-package.zip` 到 VPS
2. 解压缩文件
3. 运行安装脚本：
   - Windows: `.\install.ps1`
   - Linux: `npm install && npm run build`
4. 启动应用：
   - Windows: `.\start.ps1`
   - Linux: `npm start`

### AI 助理配置 (可选)
```bash
# 安装 Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# 下载模型
ollama pull deepseek-r1:8b

# 启动服务
ollama serve
```
